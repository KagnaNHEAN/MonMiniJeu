Mon mini-jeu




But du jeu : déplacez les carrés afin d'en faire un dessin.

Amusez-vous à bouger les carrés noirs et blancs, pour cela cliquez sur le carré concerné, puis jouer avec les boutons flèches. Une fois le mouvement placé, cliquez sur le bouton stop afin de déconnecter le carré des boutons flèches; puis recommencer avec un autre carré. Si vous désirez bouger une seconde fois un carré déjà déplacé recliquez dessus pour activer ses mouvements. Astuce : vous pouvez faire bouger plusieurs carrés en même temps, pour cela cliquez autant de carrés que vous souhaitez, puis bougez les avec les boutons flèches; et enfin pour arrêter les mouvements comme précédemment cliquez sur le bouton stop. Bon jeu !
